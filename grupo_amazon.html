Descripción del Tema principal del producto

El problema que abarca para este proyecto, es la lo que una compañia de ventas de muebles usados y nuevos, que al no contar con una guia de ventas por internet, o un catalogo 3D en su plataforma para visualizar el producto en 360º, lo cual que por conyuntura de la pandemia y movilizacion de los clientes, no pueden lograr ver el producto y tambien al administrador con respecto a su seguridad al momento que un cliente desee visualizar los muebles ofrecidos.
Que con las herramientas que nos ofrece la plataforma de AWS, dar una solucion a la demanda del cliente, y asi poder brindar toda la seguridad tanto a la visualizacion, a los muebles de 3D y al no contar con un rastreo de pedido, se le brindara la implementacion de uno, asi el cliente pueda quedar satisfecho.
Poder brindar confianza al administrador al momento de ofrecerle una seguridad de reconocimiento facial a traves de imagenes, para tanto clientes como sus empleados, tambien el cliente tenga seguridad al momento de entrega de sus envios.

Herramientas

Herramientas a Usar

Visual Studio Code

Editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web.

AWS Amplify

Es una solución completa que permite a los desarrolladores web y móviles de front-end crear, enviar y alojar aplicaciones de pila completa en AWS de forma sencilla.

GitHub

Forja para alojar proyectos utilizando el sistema de control de versiones Git.

AWS Location

Agregue mapas, lugares, búsqueda, geocodificación, geocodificación inversa, direccionamiento, seguimiento y geovallas a sus aplicaciones.

Amazon S3

Servicio de amazon basado en Buckets para almacenar archivos para uso compartido entre servicios de nubes.

AWS Lambda

Es un servicio informático sin servidor y basado en eventos que le permite ejecutar código para prácticamente cualquier tipo de aplicación.

IOT Core

Permite conectar dispositivos a servicios de AWS y a otros dispositivos, proteger datos e interacciones, procesar y actuar sobre datos de dispositivos.

AWS SNS

es un servicio administrado con el que se ofrece la entrega de mensajes de los publicadores a los suscriptores.

Amazon Lex

Es un servicio para crear interfaces conversacionales en cualquier aplicación que utilice voz y texto.

Amazon Sumerian

Es un conjunto de herramientas basadas en navegador para crear aplicaciones 3D de realidad virtual (VR) de alta calidad (AR) y realidad aumentada (AR) fácilmente sin requerir conocimientos de programación o gráficos 3D.

Amazon EC2

Permite a los usuarios alquilar computadores virtuales en los cuales pueden ejecutar sus propias aplicaciones.

Amazon Route 53

Es un servicio web de sistema de nombres de dominio (DNS) escalable y de alta disponibilidad.

SSL

es el acrónimo de Secure Sockets Layer (capa de sockets seguros), la tecnología estándar para mantener segura una conexión a Internet.

kommunicate

Es un software de soporte al cliente híbrido humano + bot para soporte proactivo, en tiempo real y personalizado.
